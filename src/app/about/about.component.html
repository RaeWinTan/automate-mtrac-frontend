Features
Remembers RAC form

Unlike the official webpage, this webpage saves previously set values of the Risk Assessment Card so you dont have to click on/type the same values each time you want to send a rac.

Change default RAC form values

You are able to change default values of the RAC for a more customised default RAC values.

No Need relogin

As long as you don't logout, you wont have to log back in each time you want to send a RAC.

Prerequisites
Account creation

To use this application you must first have an account with the official mtrac website.

Browser

This webpage saves your data in your browsers' storage. To enjoy the full functionality of this website, you must not be in incognito mode. Use the same browser from the same device so that you don't have to reset all values each time.

Initialisation

If this is your first time using this website on a different device/different browser. Here are the steps you have to take.

  • 1. Key in correct credentials and Login.
  • 2. Ensure you are in the `Send rac` page.
  • 3. Type in correct values into the RAC form and click on `SEND RAC`. This values will be set as default values for all subsequent times you send RAC unless you `Change default form values`.
How this website works
Web Api

The official mtrac website works like most modern websites. It has a frontend in a form of a webpage aka client(what users like you interect with), and a backend server (i.e. a server which transmit data either to the client or vice versa). Some clicks on the mtrac webpage (i.e. when you click the login button or the send mtrac button on the webpage) make network calls to the backend server of the mtrac website. What allows the frontend of the website to interact with the backend is called a web api, in layman's terms the brain of the website is the backend and the web api is just a way to interact with the backend. Web api is a subset of backend but for simplicity sake you can use it interchangeably throughout this article. The image below is roughly how the official mtrac application works.

Difference

This website still uses the same web api as the official mtrac webpage just that the frontend (the webpage served to the client, like a PC or phone) is different. Since this website shares the same `Brain` (web api) as a the official mtrac webpage, this webpage is compatible with the mtrac website. This website just serves a modified webpage. Read the section on 'Features' to understand the differenciating factor of this frontend compared to the official website's webpage.

Problem statement
Red tape

In the SAF there's just too many red tapes a driver must go through just to go for a drive. One of the time comsuming process that can expedite is the online MT-RAC. This webpage saves RAC form values from previous submissions so that drivers do not have to re-enter the same values each time.
